Clases

Es una "plantilla" para generar objetos

Instancias

Son los objetos generados a partir de una clase en particular

Metodos

Son funciones asociadas a un objeto

this

Es una palabra reservada que hace referencia al propio objeto en el que estoy

Visibilidad

Public -> accesible desde fuera Protected -> accesible desde dentro y desde clases padres o hijas Private -> accesible solo desde dentro (this) por la propia clase

TypeScript

SuperSet de JS -> Necesita ser transpilado a JS para funcionar
